Blog Post Webpage - Project Structure

Description

Create a blog post webpage that includes a list of recent posts and displays full post content. The page should be well-organized, visually balanced, and easy to read.

Objective

This project is designed to help you apply your knowledge of CSS Grid and semantic HTML by building a structured, responsive webpage. Through this project, you will practice:

  • Creating a structured layout using semantic HTML elements like header, main, aside, and footer.
  • Positioning elements in a grid using properties like display: grid, grid-template-columns, and grid-template-areas.
  • Controlling alignment and spacing with properties such as gap, justify-content, and align-items.
  • Making elements span across grid areas using grid-column and grid-row.

Project Practical Requirements

  • Header
    • Contains a logo labeled: MyBlog
    • Includes a navigation bar with links: Home, Posts, About, Contact
    • The logo is positioned beside the navigation
  • Aside (Left Side)
    • Displays a list of at least 3 recent posts
  • Main Content Area
    • Displays blog posts to the right of the sidebar
  • Posts
    • Each post includes a title and body
  • Footer
    • Appears at the bottom of the page

Project Layout

The layout should be implemented using CSS Grid with the following structure:

  • A grid container defining the areas: header, aside, main, and footer.
  • Use grid-template-areas to assign regions.
  • The page should form two columns:
    • Aside: ~25% width
    • Main content: ~75% width
  • The header spans the full top row; the footer spans the full bottom row.
  • Ensure good spacing and responsive behavior on smaller screens.
